本项目是浙江树人学院的期中作业，基于 Three.js 技术构建的 3D数字孪生校园交互系统。系统在浏览器中还原了校园主要建筑（教学楼、图书馆、实验楼等）、道路网络和绿化植被，用户可通过鼠标拖拽、缩放自由漫游校园，点击任意建筑查看详细信息，并支持图层切换和多种视角变换。项目旨在通过可视化技术展示校园空间布局，为智慧校园建设提供交互式数字底座。
核心功能一：三维场景漫游
用户可通过鼠标自由控制相机视角，从任意角度观察校园场景。具体操作方式为：鼠标左键拖拽可旋转视角，鼠标右键拖拽可平移场景，鼠标滚轮则用于缩放视野。该功能基于Three.js的OrbitControls组件实现，系统会实时捕获用户操作并更新相机位置，最终重新渲染画面，为用户提供沉浸式的校园浏览体验。
核心功能二：建筑信息查询
用户点击校园中的任意建筑或设施后，系统会弹出信息卡片，显示该建筑的名称和详细介绍。实现原理是：系统获取用户点击的屏幕坐标，通过Raycaster射线检测技术遍历场景中的所有可交互对象，判断是否有建筑被命中。若命中，则提取该建筑的userData中的名称和描述信息，显示在悬浮卡片中，卡片会在4秒后自动关闭。支持查询的对象包括教学楼、图书馆、实验楼、升旗台、天桥、连廊以及树木、灌木和道路等。
核心功能三：图层可视控制
用户可通过底部的控制按钮独立控制三类场景图层的显示与隐藏，方便聚焦查看特定内容。三个按钮分别对应建筑图层、植被图层和道路图层。当用户点击按钮时，系统会获取对应的图层分组（buildingGroup、vegetationGroup、roadGroup），切换其visible属性，同时按钮样式会相应变化以反映当前状态。默认状态下所有图层均为可见，用户可以自由组合开启或关闭任意图层，例如仅显示建筑以专注布局分析，或仅显示道路以查看路网结构。
